📖 About Unix Timestamps

  • Unix Timestamp: Number of seconds (or milliseconds) since January 1, 1970, 00:00:00 UTC (Unix Epoch)
  • Seconds: Standard Unix timestamp (10 digits) - most common format
  • Milliseconds: JavaScript/Java format (13 digits) - includes milliseconds
  • UTC: Coordinated Universal Time - standard time reference

💡 Common Uses

  • Database timestamps (MySQL, PostgreSQL, MongoDB)
  • API responses and requests
  • Log file analysis
  • Cache expiration times
  • Session management
  • Event scheduling
